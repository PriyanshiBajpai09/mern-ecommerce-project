# 🛍️ MERN E-Commerce Web App

✨ A modern **Full Stack E-Commerce Application** built using the powerful **MERN Stack** (MongoDB, Express, React, Node.js).
💻 Designed with a clean UI, smooth functionality, and real-world deployment.

---

## 🌐 Live Links

🔗 **Frontend:** https://mern-ecommerce-project-green.vercel.app/

⚙️ **Backend:** https://mern-backend-jozu.onrender.com

---

## ✨ Features

### 👤 User Side

* 🔐 Secure Login & Registration (JWT Auth)
* 🛒 Add to Cart Functionality
* 🔍 Search & Filter Products
* 📦 View Product Details
* 🎯 Category & Price Filtering

### 🛠️ Admin Panel

* ➕ Create Categories
* 📦 Add / Update / Delete Products
* 👥 Manage Users

---

## 🧑‍💻 Tech Stack

### 🎨 Frontend

* ⚛️ React.js
* 🎨 Bootstrap / CSS
* 🔗 Axios

### ⚙️ Backend

* 🟢 Node.js
* 🚀 Express.js

### 🗄️ Database

* 🍃 MongoDB Atlas

### ☁️ Deployment

* 🌍 Frontend → Vercel
* 🖥️ Backend → Render

## 🚀 Installation

### 📥 Clone Repo

git clone https://github.com/PriyanshiBajpai09/mern-ecommerce-project.git
cd mern-ecommerce-project


---

### 📦 Install Dependencies

npm install
cd frontend
npm install


---

### ▶️ Run Locally

#### Backend


npm run dev


#### Frontend

cd frontend
npm start
